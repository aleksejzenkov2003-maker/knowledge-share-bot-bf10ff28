## Текущие возможности

### Папки документов
- 📊 Статистика по папкам (документы, чанки, размер, ошибки)
- 🗑️ Очистка папки — удаление всех документов с подтверждением
- 🔄 Переобработка — все документы / только ошибки / ожидающие
- ⚙️ Dropdown меню с быстрым доступом к действиям

### RAG-система
- Hybrid search (FTS + embedding + web search)
- Условный web search для экономии API
- Автоматическая переобработка документов с ошибками
