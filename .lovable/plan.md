
# План исправления зависания Bitrix24 чата

## Диагностика

Обнаружено две проблемы:

1. **Неверный формат `portal_domain` в API-ключе**
   - Сохранено: `https://bitrix.artpatent.ru/marketplace/app/197/`
   - Bitrix24 SDK отправляет: `bitrix.artpatent.ru`
   - Сравнение не проходит, авторизация отклоняется

2. **Страница не показывает ошибку при проблемах с авторизацией**
   - Спиннер продолжает крутиться бесконечно
   - Пользователь не видит причину ошибки

---

## Шаг 1: Исправить данные API-ключа

Обновить значение `portal_domain` в таблице `department_api_keys`:

```sql
UPDATE department_api_keys 
SET portal_domain = 'bitrix.artpatent.ru'
WHERE id = 'cab52afe-682a-46ba-85f8-8acfc653067b';
```

## Шаг 2: Улучшить валидацию в админке

Добавить в форму создания/редактирования API-ключей:
- Валидацию формата домена (только домен, без протокола и пути)
- Подсказку/пример правильного формата
- Автоочистку URL до чистого домена

Файл: `src/pages/ApiKeys.tsx`

## Шаг 3: Улучшить логирование в edge-функции

Добавить в `handleAuth` детальное логирование:

- Какой запрос отправлен в БД
- Результат поиска

Файл: `supabase/functions/bitrix-chat-api/index.ts`

- Улучшить обработку сетевых ошибок
