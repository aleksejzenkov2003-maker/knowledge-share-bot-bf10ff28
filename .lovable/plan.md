

## Проблема

На скриншоте видно:
1. **Markdown-досье уродливое** — реквизиты в одну строку, товарные знаки как список номеров без ссылок, нет визуального разделения секций
2. **ReputationCompanyCard показывает "Без названия"** — часть данных теряется при маппинге
3. **Карточка вообще не показывает товарные знаки** (`_trademarks`) — в компоненте нет секции для интеллектуальной собственности
4. **Нет кликабельных ссылок** — номера FIPS не ведут на сайт ФИПС, ИНН/ОГРН не ведут на reputation.ru

## План изменений

### Файл 1: `src/components/chat/ReputationCompanyCard.tsx` — полная переработка

**Цель:** Превратить карточку из минимальной в полноценное досье со всеми данными.

**Изменение 1: Добавить секцию "Интеллектуальная собственность"**

Новая секция после "Контакты" — рендерит `data._trademarks`. Каждый товарный знак/патент показывается как строка с:
- Номер (кликабельный, ведёт на `https://new1.fips.ru/registers/...` для патентов или поиск по номеру)  
- Название (если есть `Name` / `Title`)
- Статус — бейдж "Действующий" (зелёный) / "Нет данных" (серый)
- Источник (`_source`: patents → "Патент", applications → "Заявка")

**Изменение 2: Сделать ИНН/ОГРН кликабельными**

ИНН → ссылка на `https://reputation.ru/company/inn-${inn}` (откроется в новой вкладке)
ОГРН → ссылка на аналогичную страницу

Вместо обычного `<span>` — `<a>` с `target="_blank"` и стилем `text-primary hover:underline`.

**Изменение 3: Улучшить отображение адреса**

Адрес — длинный текст, сейчас обрезается. Сделать многострочный с `break-words`.

**Изменение 4: Показать дату ликвидации для ликвидированных компаний**

Если есть `LiquidationDate` — показать рядом со статусом красным текстом.

**Изменение 5: Защитить от объектов во всех полях**

Добавить `safeString()` helper — если значение object, пытаться извлечь `.Name`, `.Value`, `.Text`, иначе `JSON.stringify`. Применить ко всем рендерам.

### Файл 2: `supabase/functions/chat-stream/index.ts` — улучшить markdown-форматирование

**Изменение 1: Реквизиты — в таблицу**

Вместо `ИНН: ... ОГРН: ... КПП: ...` в одну строку → markdown-таблица:

```
| Поле | Значение |
|------|----------|
| ИНН  | 1655041338 |
| ОГРН | 1021602833790 |
```

**Изменение 2: Товарные знаки — с ссылками**

Для каждого товарного знака добавить ссылку на ФИПС:
- Патенты: `https://new1.fips.ru/registers/doc/${RegistrationNumber}` 
- По номеру заявки: `https://new1.fips.ru/publication/...`

Формат: `- [№981540](https://...) — Действующий` вместо `- 981540 — Действующий`

**Изменение 3: Показывать название товарного знака**

Если у товарного знака есть `Name`/`Title` — показывать его после номера.

### Итого файлов: 2
- `src/components/chat/ReputationCompanyCard.tsx` — 5 изменений (секция ИС, кликабельные реквизиты, safe rendering)
- `supabase/functions/chat-stream/index.ts` — 3 изменения (таблица реквизитов, ссылки на ФИПС, названия ТЗ)
