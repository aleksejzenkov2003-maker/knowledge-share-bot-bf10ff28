
# Улучшение RAG-поиска для Excel-данных (МКТУ) — ВЫПОЛНЕНО

## Что сделано

### 1. ✅ Гибридный FTS: AND + OR fallback (SQL)
- `smart_fts_search` теперь сначала пробует AND-запрос
- Если AND вернул < 10 результатов, автоматически добавляет OR-результаты с пониженным рангом (x0.5)
- OR-результаты дедуплицируются относительно AND-результатов

### 2. ✅ Улучшенный keyword_search (SQL)
- Числовые ключевые слова теперь также ищут паттерны: `Класс: N`, `Класс N`, `Class: N`, `No: N`, `№ N`
- Это позволяет находить Excel-чанки МКТУ по номеру класса

### 3. ✅ Query Expansion (chat-stream)
- Словарь синонимов: МКТУ→класс/классификация, ателье→одежда/пошив, и т.д.
- Автоматическое извлечение номера класса из запроса (например "МКТУ 25" → "Класс: 25")
- Расширенные термины добавляются в FTS-запрос и в keyword fallback

### 4. ✅ Дедупликация чанков (chat-stream)
- После сбора результатов FTS/keyword, удаляются дубликаты по первым 200 символам контента
- Логирование количества удалённых дубликатов
