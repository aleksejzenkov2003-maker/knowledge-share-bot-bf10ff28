

## План: Дизайн 1-в-1 как Perplexity

Из скриншотов Perplexity видно:
- Фон страницы: очень светлый тёплый бежевый `#F5F3EE`
- Поле ввода: **белое**, с тонкой серой рамкой `#E8E6E1`, скругление `~24px`, **без подложки/контейнера** — просто плавает на фоне
- Нет разделителя, нет белой подложки под инпутом — инпут напрямую на бежевом фоне
- Шрифт: система / Inter, цвет текста `#1B1B1A`, placeholder `#9B9B95`
- Иконки в инпуте: `+` слева, модель/микрофон/send справа — всё внутри поля
- Помощь под инпутом — мелкий серый текст

### Изменения

**Файл 1: `src/index.css`**
- `--background`: обновить на бежевый `39 23% 95%` (~#F5F3EE)  
- `--foreground`: тёмный `40 5% 10%` (~#1B1B1A)
- `--chat-background`: убрать отдельную переменную, использовать `--background`
- `--muted-foreground`: `40 3% 60%` (~#9B9B95) для placeholder
- `--border`: `38 12% 89%` (~#E8E6E1) — тёплый тонкий бордер

**Файл 2: `src/components/chat/ChatInputEnhanced.tsx`**
- Убрать `bg-muted/50` с контейнера инпута → заменить на `bg-white` (белый инпут на бежевом фоне)
- Бордер: `border-[#E8E6E1]` вместо `border-border`, без изменения при фокусе (или очень мягкая тень)
- Убрать `pb-4` из внешнего контейнера — просто отступ

**Файл 3: `src/pages/Chat.tsx`**
- Убрать `bg-background` с обёртки инпута — инпут просто на том же бежевом фоне без белой подложки

**Файл 4: `src/pages/DepartmentChat.tsx`**
- Аналогично убрать `bg-background` с обёртки инпута
