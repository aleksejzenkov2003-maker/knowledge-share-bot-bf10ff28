### 3. Содержимое инструкции

**Шаг 1: В нашей админ-панели (API-ключи)**
- Создать API-ключ для каждого отдела
- Указать домен портала: `company.bitrix24.ru`
- Выбрать отдел

**Шаг 2: В Bitrix24**
1. Перейти: **Меню** → **Разработчикам** → **Другое**
2. Выбрать "локальное приложение"
3. Заполнить форму:

| Поле | Значение |
|------|----------|
| Путь вашего обработчика | `https://knowledge-share-bot.lovable.app/bitrix24-app/department.html` |
| Путь для первоначальной установки | (оставить пустым) |
| Использует только API | ❌ (не отмечать) |
| Поддерживает BitrixMobile | ❌ (не отмечать) |
| Название пункта меню | `AI Юрист` (или другое название) |
| Настройка прав | `CRM (crm)`, `Пользователи (user)` |

4. Нажать **"Перейти к приложению"** для сохранения

**Для нескольких отделов:**
- Создать отдельное локальное приложение для каждого отдела
- Использовать разные названия меню: "AI Юрист", "AI Бухгалтер", "AI HR"
- URL остаётся одинаковым — система автоматически определит отдел пользователя
