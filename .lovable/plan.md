

# Открытие раздела Reputation для всех пользователей

## Что нужно сделать

Сейчас Reputation доступен только админам. Нужно сделать его доступным всем ролям. История поисков уже изолирована по пользователям — RLS-политики на таблице `reputation_reports` фильтруют по `user_id = auth.uid()`, так что каждый видит только свои отчёты.

## Изменения

### 1. `src/components/layout/AdminSidebar.tsx`
- Убрать Reputation из `managementItems` (admin-only)
- Добавить в `mainMenuItems` (без ограничения ролей) — после "Проекты"

### 2. `src/App.tsx`
- Убрать `allowedRoles={['admin']}` с маршрута `/reputation`, оставить просто `<ProtectedRoute>`

Никаких изменений в БД или RLS не требуется — данные уже изолированы по пользователям.
