
# Выполнено: LLM Query Expansion для МКТУ

## Что сделано

В `supabase/functions/chat-stream/index.ts` добавлен **STEP 0.5: LLM Query Expansion**:

1. **Детекция МКТУ-контекста**: проверяется системный промпт и текст запроса на наличие "мкту", "mktu", "международная классификация"
2. **LLM расширение запроса**: вызывается Gemini 2.5 Flash Lite через Lovable AI Gateway для генерации 15-25 формальных терминов МКТУ из бытового описания
3. **Параллельный keyword search**: с LLM-расширенными терминами выполняется `keyword_search` (ILIKE) и результаты мержатся с FTS-кандидатами
4. **Единый reranking**: объединённый набор кандидатов (FTS + LLM keyword) отправляется в Claude reranker

### Пример работы

```
Запрос: "вендинговое оборудование"
LLM expansion → ["аппараты торговые автоматические", "машины для продажи товаров", 
                  "аппараты для приготовления напитков", "программное обеспечение для торговли", ...]
keyword_search → находит чанки с этими терминами в МКТУ
reranker → отбирает наиболее релевантные классы
```
