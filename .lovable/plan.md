
# Интерактивная инструкция (Product Tour) с driver.js

## Что будет сделано

Добавление интерактивного гида по интерфейсу в стиле driver.js -- пошаговые подсказки с подсветкой элементов, кнопками "Далее/Назад" и разделением по ролям.

## Где появится кнопка

- В правом верхнем углу (AdminHeader) -- кнопка "?" (HelpCircle) для всех пользователей
- В Bitrix-виджетах -- аналогичная кнопка в шапке

## Три набора туров

### 1. Тур для сотрудника (employee)
- Чат: как отправить сообщение, выбрать роль ассистента, прикрепить файл
- Чат отдела: переключение чатов, совместная работа
- Проекты: создание, @упоминание агентов, память проекта

### 2. Тур для администратора (admin/moderator)
- Дашборд: обзор метрик
- Роли чатов: создание/настройка ассистентов
- Папки и Документы: загрузка базы знаний, переобработка
- Промпты и Провайдеры: настройка AI
- Пользователи и Отделы: управление доступом
- Эталоны: добавление эталонных ответов
- Аудит ПДн: просмотр замаскированных данных
- API-ключи, Логи, Битрикс-сессии

### 3. Тур для Bitrix-виджета
- Как начать чат, отправить вопрос, использовать контекст

### Как работает driver.js в React

```typescript
import { driver } from 'driver.js';
import 'driver.js/dist/driver.css';

const startTour = () => {
  const driverObj = driver({
    showProgress: true,
    steps: [
      {
        element: '[data-tour="sidebar-chat"]',
        popover: {
          title: 'Чат',
          description: 'Здесь вы общаетесь с AI-ассистентом',
        }
      },
      // ...
    ]
  });
  driverObj.drive();
};
```

### Меню выбора тура (кнопка "?")

При нажатии на кнопку откроется DropdownMenu со списком доступных туров в зависимости от роли:
- Сотрудник: "Как пользоваться чатом", "Чат отдела", "Проекты"
- Админ: все вышеперечисленное + "Управление ролями", "База знаний", "Настройка AI", "Администрирование"

Каждый пункт запускает тур с навигацией на нужную страницу и пошаговой подсветкой элементов.
